# 🌱 About The Project

BloomShop is a modern full-stack web application designed for people who love decorating their homes with plants. The platform allows users to browse decorative plants, explore product details, and experience a clean and responsive shopping interface.

The project focuses on:

- Modern UI/UX design
- Responsive layouts
- Smooth shopping experience
- Full-stack architecture
- Real-world e-commerce workflow

---

# 🚀 Features

## 🛍️ User Features
